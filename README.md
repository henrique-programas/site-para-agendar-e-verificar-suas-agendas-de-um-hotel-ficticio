🏨 Sistema de Agenda e Hospedagem
📌 Sobre o Projeto

Este projeto consiste em um sistema web desenvolvido para facilitar o agendamento e hospedagem de clientes em um hotel fictício. A plataforma permite que usuários visualizem quartos disponíveis, realizem reservas e acompanhem suas hospedagens de forma prática e organizada.

Além da área do cliente, o sistema também conta com um painel administrativo para gerenciamento completo de quartos, reservas e usuários.

🎯 Objetivo

O principal objetivo deste sistema é oferecer uma solução moderna e eficiente para controle de hospedagens, automatizando processos como reservas, organização de agendas e administração interna.

🚀 Funcionalidades
👤 Área do Cliente
Cadastro de usuários
Login e autenticação
Visualização de quartos disponíveis
Consulta de datas disponíveis
Realização de reservas
Histórico de hospedagens
Cancelamento de reservas
🛠️ Área Administrativa
Dashboard com informações gerais
Gerenciamento de quartos
Cadastro, edição e exclusão de quartos
Controle de reservas
Gerenciamento de usuários
Relatórios internos
